Codice di esempio

Qui c'è un estratto dello sketch Arduino che gestisce i pulsanti, i relè e legge la temperatura via sensore DS18B20:

#include <OneWire.h>
#include <DallasTemperature.h>

const int relayPin = 7;
const int buttonPin = 2;

OneWire oneWire(10);
DallasTemperature sensors(&oneWire);

void setup() {
  pinMode(relayPin, OUTPUT);
  pinMode(buttonPin, INPUT_PULLUP);
  Serial.begin(9600);
  sensors.begin();
}

void loop() {
  bool button = digitalRead(buttonPin) == LOW;
  digitalWrite(relayPin, button);

  sensors.requestTemperatures();
  float tempC = sensors.getTempCByIndex(0);
  Serial.print("Temperatura: ");
  Serial.print(tempC);
  Serial.println(" °C");
  delay(1000);
}

Questo sketch fa il debounce semplice dei pulsanti e invia la temperatura sul monitor seriale, mentre il relè accende/spegne le luci.
