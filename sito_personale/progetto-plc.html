Codice di esempio

Nel PLC ho usato una logica a gradini (ladder) con un blocco di interblocco e un blocco di emergenza. Di seguito un esempio semplificato (pseudocodice ladder):

|--[ START ]-----[ ]----( )---|
|   |  PB1      |           |
|   |          ( )----------|
|   |           M1          |
|   |                       |
|   |--[ NOT STOP ]-[ ]-( )-|
|           |          Q1   |
|           |               |
|--[ EMERGENCY ]-[ ]--( )--|
|      |    |     |         |
|      |    |     Q2        |
|      |    |
|      |    +--[ ]--( )-----|
|      |       PB2   Q3     |

In pratica, il pulsante START abilita l'uscita con interblocco e il STOP interrompe immediatamente. L'uscita Q2 è legata all'arresto di emergenza.
