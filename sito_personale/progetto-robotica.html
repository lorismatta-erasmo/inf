Codice di esempio

Un semplice script Python per generare una traiettoria lineare per un braccio robotico e stampare gli angoli delle giunture:

import numpy as np

def linear_trajectory(start, end, steps=20):
    return np.linspace(start, end, steps)

# Esempio: punto iniziale (x,y) -> punto finale (x,y)
start = np.array([0.2, 0.1])
end = np.array([0.5, 0.4])

for point in linear_trajectory(start, end):
    # Qui si calcolano gli angoli con la cinematica inversa (es. 2DOF)
    # angle1, angle2 = inverse_kinematics(point)
    # print(f"Angoli: {angle1:.2f}, {angle2:.2f}")
    print(f"Punto: {point[0]:.2f}, {point[1]:.2f}")

Questo approccio aiuta a capire come trasformare un percorso desiderato (in coordinate cartesiane) in comandi per i motori del braccio.
